# Fundamentos da Linguagem GNU Octave

Esse repositório reúne uma coleção de códigos que demonstram os fundamentos básicos do GNU Octave. Inclui exemplos como:

- Hello World
- Cálculos básicos
- Variáveis
- Input
- Strings
- Vetores e Matrizes
- Switch Case
- Gráficos básicos, especiais e em 3D.
- Funções
- If and Else
- Switch Case
- Laços
- Manipulação de arquivos
